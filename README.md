# FindPlace
  - 사진의 메타데이터(EXIF)를 기반으로 사진의 다양한 정보를 활용한 앱
  
# Used
  ### 1. Android Studio 4.2 (API Level 28)
  ### 2. Kotlin 1.3.72
    1. ExifInterface
    2. Kakao Map API with GeoCoder
    3. Kakao Local API (예정)
